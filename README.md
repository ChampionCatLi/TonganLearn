## 接入指南
 
### 一.使用Gradle方式:

#### 1.在Project的build.gradle文件中配置。

```
allprojects {
    repositories {
        //必须配置
        jcenter()
    }
}
```

#### 2.在App的build.gradle的文件中添加依赖。

```
dependencies {
        implementation 'com.tongan_study:tongan_learn_library:1.1.1'
}
```
